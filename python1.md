## 2. 변수와 자료형

### A. 변수(Variable)란?

- 변수란 데이터를 저장하는 공간 (이름이 붙은 저장소)
- 변수는 값을 저장할 수 있으며, 언제든 변경 가능
- 파이썬에서는 자료형을 별도로 선언할 필요 없이, 값을 할당하면 자동으로 타입이 지정
- `=` 기호를 사용하여 값을 할당

### B. 자료형이란?

- 변수에 저장할 수 있는 데이터의 종류를 **자료형(data type)** 이라고 함
- 파이썬의 기본 자료형은 다음과 같음

| 자료형           | 설명                          | 예제                                  |
| ------------- | --------------------------- | ----------------------------------- |
| `int` (정수)    | 정수 값을 저장하는 자료형              | `x = 10`                            |
| `float` (실수)  | 소수점이 있는 숫자                  | `y = 3.14`                          |
| `str` (문자열)   | 문자나 문자열을 저장                 | `name = "Python"`                   |
| `bool` (불리언)  | 참(True) 또는 거짓(False)을 저장    | `is_student = True`                 |
| `list` (리스트)  | 여러 개의 값을 저장하는 변경 가능한 자료형    | `fruits = ["사과", "배", "포도"]`        |
| `tuple` (튜플)  | 여러 개의 값을 저장하지만, 변경 불가능한 자료형 | `colors = ("red", "blue", "green")` |
| `dict` (딕셔너리) | 키-값(key-value) 쌍으로 데이터를 저장  | `student = {"이름": "철수", "나이": 20}`  |
| `set` (집합)    | 중복을 허용하지 않는 데이터 저장          | `numbers = {1, 2, 3, 3, 4, 5}`      |

## 3. 리스트, 튜플, 딕셔너리, 집합

### A. 리스트란?

- 여러 개의 값을 저장할 수 있는 자료형
- 값(요소)들을 순서대로 저장, **인덱스**를 통해 접근[^1]
- 값을 추가/변경 가능
- 대괄호 `[ ]`를 사용하여 정의

[^1]: **인덱스란?** 데이터가 저장된 위치를 나타내는 번호. 리스트, 튜플, 문자열 등은 순서(인덱스)를 가진 데이터 구조로, 특정 요소에 접근할 때 인덱스를 사용할 수 있음. 리스트에서 인덱스는 0부터 시작하며 음수도 사용 가능. 리스트는 요소 변경이 가능하지만, 튜플이나 문자열은 요소 변경이 불가능.

#### 리스트 주요 메서드

| 메서드             | 설명                    | 예제                        |
| --------------- | --------------------- | ------------------------- |
| `.append(x)`    | 리스트 끝에 요소 추가          | `fruits.append("망고")`     |
| `.insert(i, x)` | 특정 위치 i에 요소 x 추가      | `fruits.insert(1, "복숭아")` |
| `.remove(x)`    | 리스트에서 첫 번째로 등장하는 x 제거 | `fruits.remove("포도")`     |
| `.pop(i)`       | 인덱스 i 위치의 요소를 제거하고 반환 | `fruits.pop(2)`           |
| `.sort()`       | 리스트 요소를 오름차순 정렬       | `numbers.sort()`          |

#### 인덱스 사용 예시

```python
fruits = ["사과", "배", "포도", "딸기"]

# 인덱스로 접근 (0부터 시작)
print(fruits[0])   # 첫 번째 요소: "사과"
print(fruits[2])   # 세 번째 요소: "포도"

# 음수 인덱스 (뒤에서부터 접근)
print(fruits[-1])  # 마지막 요소: "딸기"
print(fruits[-2])  # 뒤에서 두 번째 요소: "포도"
```

```python
# 튜플에서 인덱스로 접근하기
colors = ("빨강", "파랑", "초록")
print(colors[0])   # "빨강"
print(colors[2])   # "초록"
```

```python
# 리스트 요소 변경
fruits = ["사과", "배", "포도"]
fruits[1] = "복숭아"   # 두 번째 요소 변경
print(fruits)          # ['사과', '복숭아', '포도']

# 문자열은 변경 불가
text = "Hello"
text[0] = "h"   # 오류 발생! 문자열은 변경 불가
```

### B. 튜플

- 리스트와 유사하지만, **수정 불가능**
- 소괄호 `( )`를 사용하여 정의
- 변경할 필요가 없는 데이터를 저장할 때 사용

---

### C. 딕셔너리

- 키-값(key-value) 형태로 데이터를 저장하는 자료구조
- 중괄호 `{ }`를 사용하며, 키를 통해 값을 빠르게 조회할 수 있음
- 키(key)는 중복 불가능, 값(value)은 중복 가능

#### 딕셔너리 주요 메서드

| 메서드          | 설명                   | 예제                        |
| ------------ | -------------------- | ------------------------- |
| `.keys()`    | 모든 키 반환              | `student.keys()`          |
| `.values()`  | 모든 값 반환              | `student.values()`        |
| `.items()`   | 모든 키-값 쌍 반환          | `student.items()`         |
| `.get(k, d)` | 키 k의 값을 반환, 없으면 d 반환 | `student.get("이름", "없음")` |

---

### D. 집합(set)

- 중괄호 `{ }`를 사용하여 정의하며, **중복을 허용하지 않음**
- 순서가 없기 때문에 인덱스로 접근할 수 없음
- 교집합, 합집합, 차집합 연산 수행 가능
