# 2주차: 연산자(operator)

**연산자**: 값과 값을 계산하는 기호

| 연산자 종류 | 설명 | 예제 |
|------------|------|------|
| **산술 연산자** | 숫자 값을 연산 (사칙연산 등) | `+`, `-`, `*`, `/`, `%`, `//`, `**` |
| **비교 연산자** | 두 값을 비교 (True/False 반환) | `==`, `!=`, `>`, `<`, `>=`, `<=` |
| **논리 연산자** | 논리값을 조합 (True/False 반환) | `and`, `or`, `not` |
| **대입 연산자** | 변수에 값을 할당 | `=`, `+=`, `-=`, `*=`, `/=`, `//=`, `%=`, `**=` |
| **멤버십 연산자** | 특정 값이 리스트 등에 포함 여부 확인 | `in`, `not in` |

---

## 1. 산술연산자(arithmetic operators): 숫자계산

| 연산자 | 설명 | 예제 | 결과 |
|--------|------|------|------|
| `+` | 덧셈 | `5 + 3` | `8` |
| `-` | 뺄셈 | `5 - 3` | `2` |
| `*` | 곱셈 | `5 * 3` | `15` |
| `/` | 나눗셈 (소수점 포함) | `5 / 2` | `2.5` |
| `//` | 나눗셈 (몫만 반환) | `5 // 2` | `2` |
| `%` | 나눗셈 (나머지) | `5 % 2` | `1` |
| `**` | 거듭제곱 | `2 ** 3` | `8` |

```python
a, b = 10, 3

print(a + b)   # 13
print(a - b)   # 7
print(a * b)   # 30
print(a / b)   # 3.3333
print(a // b)  # 3 (몫)
print(a % b)   # 1 (나머지)
print(a ** b)  # 1000 (10^3)
```

---

## 2. 비교연산자(comparison operators): 두 값을 비교하여 참(True) 혹은 거짓(False)을 반환

| 연산자 | 설명 | 예제 | 결과 |
|--------|------|------|------|
| `==` | 두 값이 같으면 True | `5 == 5` | `True` |
| `!=` | 두 값이 다르면 True | `5 != 3` | `True` |
| `>` | 왼쪽 값이 크면 True | `5 > 3` | `True` |
| `<` | 오른쪽 값이 크면 True | `5 < 3` | `False` |
| `>=` | 왼쪽 값이 크거나 같으면 True | `5 >= 5` | `True` |
| `<=` | 오른쪽 값이 크거나 같으면 True | `5 <= 3` | `False` |

```python
x, y = 5, 5
print(x == y)  # True
print(x != y)  # False
print(x > 3)   # True
print(y <= 2)  # False
```

---

## 3. 논리연산자(logical operators)

| 연산자 | 설명 | 예제 | 결과 |
|--------|------|------|------|
| `and` | 모든 조건이 True면 True | `(5 > 3) and (10 > 5)` | `True` |
| `or` | 하나라도 True면 True | `(5 > 3) or (10 < 5)` | `True` |
| `not` | 논리값을 반대로 변경 | `not(5 > 3)` | `False` |

```python
a, b, c = 10, 5, 3

print(a > b and b > c)   # True
print(a > b or b < c)    # True
print(not a)             # False
```

---

## 4. 멤버십 연산자(membership operators)

| 연산자 | 설명 | 예제 | 결과 |
|--------|------|------|------|
| `in` | 값이 포함되어 있으면 True | `"a" in "apple"` | `True` |
| `not in` | 값이 포함되지 않으면 True | `"x" not in "hello"` | `True` |

```python
fruits = ["사과", "배", "포도"]
print("사과" in fruits)      # True
print("딸기" not in fruits)  # True
```

---

## 5. 대입연산자(assignment operators)

| 연산자 | 설명 | 예제 | 결과 |
|--------|------|------|------|
| `=` | 변수에 값 할당 | `x = 10` | `x = 10` |
| `+=` | 더한 후 저장 | `x += 5` | `x = x + 5` |
| `-=` | 뺀 후 저장 | `x -= 3` | `x = x - 3` |
| `*=` | 곱한 후 저장 | `x *= 2` | `x = x * 2` |
| `/=` | 나눈 후 저장 | `x /= 2` | `x = x / 2` |
| `//=` | 몫을 저장 | `x //= 2` | `x = x // 2` |
| `%=` | 나머지를 저장 | `x %= 2` | `x = x % 2` |
| `**=` | 거듭제곱 후 저장 | `x **= 3` | `x = x ** 3` |

```python
x = 10
x += 5   # x = x + 5  →  15
x *= 2   # x = x * 2  →  30
x //= 4  # x = x // 4 →  7
```
